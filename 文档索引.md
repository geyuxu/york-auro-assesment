## 🎯 快速开始

### 1. 首次建图
```bash
# 参考 aaa 文件中的命令
cat aaa
```

### 2. 继续建图（从保存的状态）
```bash

bash 加载Cartographer状态.sh
```

### 3. 保存进度
```bash
# 建图过程中随时保存
bash 保存Cartographer状态.sh robot1 我的地图
```
